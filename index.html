Ap-statistics

This is a web application that displays the popularity and win rates of the new AP items before and after the rework.

Download Download View Ap-statistics on GitHub

This is a website designed by high school students entering college in the fall. With our newfound experience in web design, we have created a tool that displays the changes in big AP item usage between the 5.11 and 5.14 patches. There is a graph that is meant to display the two pick rates. However, we had issues with the rate limit and did not have time to figure out how to have the program finish executing before 40000 matchIds were analyzed.
